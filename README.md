# 🎲 Dice Game

A simple browser-based dice game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Random dice roll for two players
- Displays winner dynamically
- Simple and clean UI
- Refresh to play again

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 🎮 How to Play
1. Open the website
2. Refresh the page
3. Dice will roll automatically
4. Player with higher number wins
